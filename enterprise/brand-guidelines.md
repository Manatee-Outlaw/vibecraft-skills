## Applying this brand

When producing any artifact, apply these rules without being asked:
1. Use brand colours for all visual elements — headings, accents, backgrounds
2. Use specified fonts at the sizes above
3. Write in the brand voice
4. When in doubt: less is more. Clean, spacious layouts over busy ones.

If you don't have access to a specific font, use the closest available system font and flag it.
